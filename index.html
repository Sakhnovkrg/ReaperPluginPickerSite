Installation

Copy the .dll file to the UserPlugins folder inside your REAPER resource path.

To find it: Options > Show REAPER resource path in explorer

Restart REAPER — Plugin Picker will appear in the Extensions menu.

You can assign a hotkey (e.g. F1) via Actions > Show Action List... — search for Plugin Picker: Toggle.

Customization

Edit config.json to adjust colors and layout:

%APPDATA%\PluginPicker\config.json

Installation

Copy the .dylib file to the UserPlugins folder inside your REAPER resource path.

To find it: Options > Show REAPER resource path in explorer

Remove the quarantine flag (macOS blocks unsigned third-party libraries by default):

xattr -d com.apple.quarantine ~/Library/Application\ Support/REAPER/UserPlugins/reaper_plugin_picker-arm64.dylib

macOS may also ask for screen recording permission — this is needed for the screenshot capture feature.

Restart REAPER — Plugin Picker will appear in the Extensions menu.

You can assign a hotkey (e.g. F1) via Actions > Show Action List... — search for Plugin Picker: Toggle.

Customization

Edit config.json to adjust colors and layout:

~/Library/Application\ Support/PluginPicker/config.json
